##### 装饰器模式基础概念
    适用于：
        拓展一个类的功能；
        动态给对象添加功能，并且动态撤销。
    优点：
        继承的有力补充，不改变原有对象的情况下给对象拓展功能；
        通过使用不同的装饰类、不同的组合方式，实现不同的效果。
        符合开闭原则。
    缺点：
        增加程序复杂性；
